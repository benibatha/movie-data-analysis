Overview
This project analyzes a movie dataset to explore how audience preferences and movie popularity have changed over time.

The analysis focuses on:
- Genre trends across decades
- The relationship between budget and popularity
- Language distribution in movies over time

Tools & Technologies
- Python (Pandas, Matplotlib)
- SQL (SQLite)
- Jupyter Notebook

Key Insights
- Genre distribution remains relatively stable, with Drama, Comedy, and Action dominating.
- High-budget films tend to achieve higher popularity, especially in recent decades.
- English-language films dominate, but other languages are gradually increasing in share.

Dataset sourced from Kaggle (not included in repository)
